# MARKET VERDICT - 開発ログ
> 一流ストラテジストの思考を組み込んだ市場判断システム

## 🎯 コンセプト

従来のカテゴリ別表示（Fed政策、流動性など）は「材料」に過ぎない。
投資家が本当に知りたいのは「結論」（株は買いか売りか）。

---

## 🧠 設計哲学（2026-01-23確定 - 最重要！）

**「誰の手法を使うか」ではなく「何を測るか」で設計する**

### 参考にする一流の思考

| 投資家 | 核心思想 | 名言 | VERDICT反映 |
|--------|----------|------|-------------|
| **Stanley Druckenmiller** | 流動性が全てを動かす | "Liquidity drives everything" | 流動性スコア (35%) |
| **Ray Dalio** | マクロ経済サイクル | "Debt cycle determines everything" | サイクルスコア (25%) |
| **Howard Marks** | 振り子の極端を避けよ | "You can't predict. You can prepare." | センチメント (15%) |
| **Michael Howell** | 流動性の定量化 | Capital Wars著者 | Net Liq計算式 |

### 共通エッセンス = VERDICTの魂

```
┌─────────────────────────────────────────────────────────┐
│  「相場の方向性は流動性が決める」(Druckenmiller, Howell) │
│  「いつ動くかはサイクルの位置で決める」(Dalio)           │
│  「どこまで動くかは極端度で判断する」(Marks)            │
└─────────────────────────────────────────────────────────┘
```

## 📊 4本柱の構成

```
MARKET VERDICT スコア = 
    流動性スコア (35%)    ← Net Liquidity, Reserves, M2
  + サイクル位置 (25%)    ← イールドカーブ、失業率、製造業指標
  + テクニカル (25%)      ← MA乖離、RSI、52週位置

```

## 🔒 設計原則

1. **シンプルに始めて改良** - 完璧より動くもの優先
2. **透明性** - なぜその判定かを説明できる（Phase 4で強化）
3. **謙虚さ** - VERDICTは「答え」ではなく「会話のきっかけ」
4. **検証可能** - 過去データで妥当性を確認できる（Phase 5）

---

## ⚠️ 重要な注意点

### 投資助言にならないよう注意
- ❌「今すぐ買うべき」→ ✅「この環境では株式に追い風」
- ❌「来月は上昇する」→ ✅「過去のパターンでは上昇傾向」
- 個別銘柄への言及は避け、セクター/資産クラスレベルに留める

### 必須の免責表示
```
⚠️ 本分析は情報提供を目的としたものであり、
投資助言ではありません。投資判断は自己責任で行ってください。
```
